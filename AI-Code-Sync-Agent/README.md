# AI Code Sync Agent

## Features

- Extract AI generated code
- Create folders automatically
- Create files automatically
- Update existing files
- Backup before overwrite
- Git Integration
- Project Detection
- Project Runner
- Error Analyzer
- Auto Retry
- Live Sync
- SQLite Storage
- Logging
- Cache Manager
- Security Scanner

---

## Supported Projects

- Python
- React
- Next.js
- Node.js
- Java
- .NET
- Flutter
- Rust
- Go

---

## Installation

```bash
git clone <repo>

cd AI-Code-Sync-Agent

pip install -r requirements.txt

playwright install
```

---

## Run

```bash
python start.py
```
